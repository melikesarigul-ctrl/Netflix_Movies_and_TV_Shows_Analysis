# Netflix Movies & TV Shows Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset from Kaggle.
The analysis was performed using Excel, SQL, and Tableau.

## Dataset
Source: Kaggle - Netflix Movies and TV Shows Dataset  

## Tools Used
- Microsoft Excel (Data Cleaning & Pivot Tables)
- SQL (Data Analysis & Aggregations)
- Tableau (Data Visualization & Dashboard)

## Key Analysis Performed
- Number of contents by country
- Content distribution by type (Movie vs TV Show)
- Content added per year
- Most common genres
- Rating distribution analysis
- Duration analysis
