# Netflix Movies & TV Shows Analysis (1925-2021)

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset from Kaggle.
The analysis was performed using Excel, SQL, and Tableau.

## Dataset
Source: Kaggle - Netflix Movies and TV Shows Dataset (https://www.kaggle.com/datasets/shivamb/netflix-shows)

## Tools Used
- Microsoft Excel (Data Cleaning & Pivot Tables)
- SQL (Data Analysis & Aggregations)
- Tableau (Data Visualization & Dashboard)
  
## Key Analysis Performed
- Number of contents by country
- Content distribution by type (Movie vs TV Show)
- Content added per year
- Most common genres
- Rating distribution analysis
- Duration analysis

## Key Insights
- Top Countries: United States leads in content production, followed by India and the UK.  
- Popular Genres: Drama and Comedy dominate Netflix content.  
- Netflix content growth peaked between 2018–2020, showing a trend toward international expansion. 
- Distribution by Type: Movies dominate the dataset, but TV Shows have significantly increased in recent years.
- Rating Trends: Mature content (TV-MA) is the most frequent rating.
- Longest movies generally fall between 2–3 hours; TV shows mostly have 1–3 seasons.
